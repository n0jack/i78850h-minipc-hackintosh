# i78850h-minipc-hackintosh
I78850H迷你主机安装黑苹果系统
## 硬件配置
* 主板：CM238芯片组
* 处理器：i7-8850H(Coffee Lake)
* 内存条：玖合 DDR4 2666 16G*2
* SSD：西数 SN750 500GG
* 网卡：BCM94352z (DW1560) 

## Bug

- 深度睡眠唤醒后部分软件不正常工作（浏览器、预览、视频播放、VSCode等）

## 支持系统

- macOS Catalina 10.15.4 (19E266) （目前使用）
- macOS Mojave （之前使用）
